常见问题

1、除了筛选未录入的，还可以筛选其它条件吗？

可以。不管是否已录入的成绩都可以筛选出来进行编辑后一键导入。 例如，我们要录入某一科目的学生成绩，可以通过科目编号或科目名 筛选出成绩列表，然后导出excel编辑！

2、上传之前要将表格另存为？

我们目前只支持Excel 97-2003的 *.xls 和 *.xlsx 文件，上传其它版本格式有可能解析失败！

3、导入的表格字段可以部分为空吗？

不可以。如果有空数据可能会影响成绩导入的准确性！
